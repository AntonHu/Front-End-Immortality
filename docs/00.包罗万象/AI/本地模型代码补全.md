# vscode免费AI Coding环境搭建

现在层出不穷的AI IDE，Cursor，Winsurf，Trae等，对开发效率有不小的助力，但是除了Trae的限时免费，其他AI IDE都是要收费的，这无疑也是增加了研发成本。

经过一番调研测试，贫穷的我终于找到了免费使用AI编程的办法，通过部署本地模型，并分别接入CodeGPT和Cline实现完整的AI编程功能。
